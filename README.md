# Currency Converter App

تطبيق ويب يعرض أسعار العملات ويتيح التحويل باستخدام Twelvedata، مع واجهة داكنة وأيقونات مخصصة.

## المتطلبات
- حساب ومفتاح API من Twelvedata
- أيقونات العملات متاحة عبر روابط GitHub Raw (كما في مشروعك)

## الإعداد
1. انسخ الملفات إلى مستودع GitHub.
2. افتح `script.js` وضع مفتاحك في `API_KEY`.
3. تأكد من صحة أسماء ملفات الأيقونات وفقًا لقائمتك.
4. شغّل محليًا بفتح `index.html` أو انشر عبر GitHub Pages:
   - Settings → Pages → اختر الفرع `main` والمجلد `root`.

## الميزات
- تحديث الأسعار كل 20 دقيقة في طلب واحد (batch).
- بطاقات أسعار مع مؤشر اتجاه (ارتفاع/انخفاض).
- تحويل فوري بين أي عملتين بدعم سعر السوق المتوسط.
- وضع داكن/فاتح/تلقائي مع حفظ الإعداد.
- أيقونات العملات تُجلب من روابط GitHub Raw.

## ملاحظات
- تأكد من صحة ملفات الأيقونات (خاصة `egbx` مقابل `EGP`) وعدّل الخريطة في `ICONS_RATES` عند الحاجة.
- يمكنك إضافة المزيد من الأزواج في `buildRateSymbols()`.
